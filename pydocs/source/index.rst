*****************
XmsGrid |version|
*****************

**Last Updated:** January 14, 2019

The xmsgrid library is a grid library for other aquaveo products. This library contains classes used to create grids.

Examples
========

The examples for this library can be found in the examples folder in the `github repository <https://github.com/Aquaveo/xmsgrid>`_

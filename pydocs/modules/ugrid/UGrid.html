set_cellstream(self: xmsgrid.ugrid.UGrid, cellstream: iterable) → bool¶

Set the ugrid cells for the entire UGrid using a cell stream. The cells are defined into two groups. First hexahedrons, polygons, quads, triangles contain the cell type (ElemTypeEnum), number of points, and point numbers. They are generally 0-based, CCW, bottom, then top. Not true for pixel or voxel. Second polyhedrons contain the cell type, number of faces, [num points in face, point numbers (0-based, CCW when looking in)] repeated for each face.

Parameters:cellstream (iterable) – A stream of cells.
Returns:If successfully set.
Return type:bool
